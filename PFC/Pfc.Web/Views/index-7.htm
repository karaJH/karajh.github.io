常问问题

选择以下常见问题之一，以了解有关购买，出售和出租房地产的更多信息。另外，开始考虑在进入房地产搜索时要考虑的重要事项。

购买常见问答

泰国土地所有权有哪些？

永久产权(Chanote或者Nor Sor 4 Jor) 是最普遍和最值得信赖的土地产权。尽管如此，Nor Sor 3 Gor以及Nor Sor 3产权（这类产权只要准备好完整的产权，可以享受和永久产权相同权利）也可以持有、买卖以及借贷抵押，只是面积测量不如永久产权来的精确。与此同时，也有些土地所有权类型不允许业主进行买卖，因此不是真正意义上的产权所有。

泰国的测量单位有哪些？

在房屋测量上使用公制单位，在土地测量上使用佤或者莱。1莱=1600平方米，1平方佤=4平方米

非泰国国籍人士可以拥有泰国的土地吗？

任何非泰国国籍人士不可以在泰国有合法土地所有权。但是外籍人士可以有以下合法权利：

  1. 以泰国公司的名义享受土地所有权。泰国公司里泰国籍人必须是多数权益控股股东。
  2. 租借土地最高长达30年
  3. 购买区别于土地的其他所有权。如，房屋别墅等等。
  4. 可供外籍人士可以合法拥有的公寓。

使用泰国公司拥有产权的优势

泰国公司在土地所有权上拥有和泰国公民一样的权利。所以土地可以转售、分割及贷款。

使用泰国公司拥有土地的注意事项

建立和经营一家公司需要时间和资金。外籍人士在股权控制方面，最多不可超过49%，所以至少51%的股权掌握在泰国人手中。相关协议需要妥善保存（最好交由合适的律师保管）来保障你的地位及对公司的绝对掌控权。如果泰国公司不再经营，可以关闭并进行资产清算。

租赁物业的优势？

租赁物业比购买物业会便宜很多。支付方式也有很大的商议空间，可以年付甚至月付，从而节约了承租者的资金。因此商业经营很多会选择这种方式。

租赁物业有哪些注意事项？

不像永久产权，租赁合约最终有失效期。与此同时，当承租人死亡后，租赁合同便自动失效。租赁物业更难以估价和出售，因为泰国银行绝大多数不会接受租赁权作为抵押而提供贷款。承租人也无权分割及售卖土地，因此也更难用租赁权来作为抵押来贷款。

如何确保租赁权的正当合法性？

首先，让值得信赖的律师跟你一起签订租赁合同或者至少阅读一遍租赁合同。租赁权可以最长拥有30年租期，所有超过3年的租赁合同必须在土地管理局进行注册。

泰国通常的做法是出租人和承租人签订3年的合同，并且可以选择续租。这样的做法省去了去土地管理局注册的时间，资金以及报税等麻烦。

当然也有土地所有人签订30年的租赁契约，并且以30+30+30年的契约方式续约。但是最近最高法院宣布这种做法不合法，并且予以废除。如果土地所有人最终没有履行30+30+30年的契约，你的相关权益将得不到法律保障。

外籍人士如何拥有产权公寓？

栋楼房首先要在土地管理局注册成“公寓房”，并且49%的“私有待售面积”可供出售给外籍所有者。外籍购房者必须符合《泰国公寓执行法》下的“外籍产权所有者”的资格。欲拥有该项资格，外籍人士需填写相关文件（外国交换交易表）表明何时何地“外国资金”将进入泰国。外籍人士要获得在泰国的产权公寓，这2项主要障碍需要得到解决。

公寓长期租赁是否可行？

公寓可以最长租赁30年，也可以续租。有些开发商提供30+30+30年的租赁契约，但是如果最终开发商不履行契约，你的第二个或者第三个30年的续约可能无法有效保障。

租赁常见问答

租赁还是购买更好？

这个完全看你的处境、需要和目的。但是不管是在泰国租赁或者购买物业，一个好的房产经纪人能够让你有完全不一样的体验。

我应该有多少预算？

以一年租期为例，在普吉岛，一个不错的一室一厅的公寓，大概是10000-30000泰铢/月起+其他公共设施费用。一套不错的三室一厅的私人泳池别墅大概在60000-100000泰铢/月+其他公共设施费用。

支付方式？

二手房 一般缴纳100000-200000泰铢的定金，余款30天内付清。但是多数二手房东可以商议协调支付方式。

开发商 一般缴纳10%的定金，剩下的余款在4个月内缴清。

外籍人士可以在泰国贷款吗？

可以，但仅限于购买产权公寓，费用以及附加费用很高。

在泰国如何给房子估价？

跟世界其他地方一样，通过比较同一区域类似的房子售价及卖房收入即可得知。由于普吉岛的市场有时可能出现不景气而且价格透明度是也是一个阻碍，我们更倾向于使用投资回报率（ROI）来评估房屋价值。通常业主的期望净回报率可以达到5%-7%。

修缮及保养

保养一套房子需要做好不少工作，尤其在普吉。普吉房地产公司会为业主提供保养维护套餐服务，并且很多别墅物业以及公寓也会使用我们公司提供的服务。

搬迁

搬迁通常不是一件轻松的事，所以我们可以推荐一些搬家公司提供一条龙服务。但是普吉岛很多的房源都是家具全配或者至少有家具配备的套餐可供选择。
